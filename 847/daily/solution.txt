[[1,2,3],[0],[0],[0]]
0th node is connected to [1,2,3] nodes
1st node is connected to [0] nodes
2nd node is connected to [0] nodes
3rd node is connected to [0] nodes

берем каждую ноду и помещаем их в очередь - {длина, нода, ноды которые были посещены - битовая маска}
изначальная длина 0, т.к. стартуем из этой вершины, помечаем что эта нода уже была посещена.

далее разгребаем очередь:
1. берем ноду сверху и проверяем, все ли вершины были уже посещены, если да то это ответ.
2. проверяем все вершины которые соединены с текущей
3. для этого обновляем текущую маску и проверяем есть ли уже такая в поиске, если есть - не учитываем,
4. если нет, то добавляем в список поиска текущую ноду и в очередь текущее сочетание - нода+маска+длина
